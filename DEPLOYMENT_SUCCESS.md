后端 API 运行正常，但通过云服务器安全组的限制，外部无法直接访问端口 3001。

前端会通过 Nginx 反向代理自动调用后端 API（路径：`/api/v1/*`），所以所有功能都正常！

### 📝 技术架构

```
Internet (Port 80)
   │
   ▼
Nginx (反向代理)
   │
   ├─▶ / (前端 → localhost:3000)
   │
   └─▶ /api/* (后端 → localhost:3001)
```

### 🔧 常用命令

```bash
# 查看服务状态

pm2 logs blog-frontend

# 重启服务
pm2 restart blog-backend
pm2 restart blog-frontend

# 查看 Nginx 状态
systemctl status nginx

# 重启 Nginx
systemctl restart nginx
```
